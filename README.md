# zh-ayxty-guide

本项目是一个 HTML 页面归档仓库，用于存放和发布多个独立的静态网页文件。这些页面涵盖不同主题和用途，彼此之间没有直接的域名或站点关联。

## 项目简介

本仓库的主要目的是作为 HTML 页面的集中存放与版本管理空间。每个页面均为独立的 HTML 文档，可能包含样式、脚本等前端资源。仓库本身不指向任何特定网站或在线服务，仅用于归档和发布。

## 目录说明

```
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各类独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源文件（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要页面目录，每个文件为一个独立 HTML 页面。
- `assets/`：存放页面引用的公共资源，如样式表、脚本和图片。

## 页面归档说明

- 所有页面均为独立完整的 HTML 文件，可单独打开或部署。
- 页面之间没有依赖关系，每个页面均可独立运行。
- 页面内容涉及多种主题，具体用途见页面内说明。
- 归档页面可能不定期更新或补充，不保证所有页面长期可用。
